**目标**: 可发布的 MVP 版本，核心功能完善
**最终目标**: 公开发布

---

## 🎯 核心功能（MVP v1）

### 必须实现的功能：

1. **用户输入问题** - 简洁的首页，用户输入困惑/问题
2. **AI智能推荐牌型** - 根据问题类型自动推荐合适的塔罗牌阵
3. **流畅抽牌体验**：
   - 3D翻牌动画
   - 洗牌效果
   - 触感反馈
4. **AI解析**（核心差异化）：
   - 结合用户问题
   - 每张牌的含义
   - 整体解读
   - 建议和指引
5. **历史记录** - 本地存储（localStorage），无需登录
6. **精美UI** - 专业、神秘感、易用

### 暂缓到 v2：

- 用户注册登录
- 用户反馈表单
- 社交分享
- 多语言支持
- 微信小程序版本

## 🛠️ 技术栈

### 前端

- **框架**: Next.js 14 (React) - App Router
- **语言**: TypeScript
- **样式**: Tailwind CSS + Framer Motion
- **状态管理**: Zustand

### AI服务

- **提供商**: DeepSeek
- **API Key**: 已配置在 .env.local
- **安全**: 仅在服务端使用

### 部署

- **平台**: Vercel（免费）
